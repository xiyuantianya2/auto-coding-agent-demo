# 网页连连看（Link Game）— 架构说明

## 项目概述

浏览器内可玩的 **连连看**：棋盘上分布成对图标，玩家依次选中两个相同图案；若存在 **不超过两次拐弯** 的空白连通路径（经典规则），则消除这一对；全部消完即过关。  
第一版包含：**计时**、**提示一步**、**多关卡**（顺序或选关）。无需登录与后端，逻辑与状态均在客户端完成。

> 仓库内 `hello-nextjs/` 为历史「故事转视频」示例；其架构说明若需对照，可从 **git 历史** 中查看此前的 `architecture.md` 版本。

---

## 核心规则（实现约束）

- 棋盘为 **二维网格**，每格为「空」或「某种图案 ID」。
- 两次点击若图案相同且 **可连**（路径沿上下左右，转弯 ≤2 次，且路径经过的格子为空；边界外视为空，与常见实现一致），则两格清空。
- **洗牌 / 生成**：保证成对出现；生成后在固定重试上限内校验 **全盘可解**（`isBoardFullySolvable`），未通过则重洗，避免返回仅局部可连、无法消完的死局。

---

## 功能模块

| 模块 | 说明 |
|------|------|
| 关卡 | 配置：行列数、图案种类数或关卡 ID；多关可顺序解锁或列表选关。 |
| 游戏状态 | 棋盘、选中项、进行中 / 胜利、当前关卡索引。 |
| 计时 | 开局计时，胜利或暂停时停止（暂停为可选增强）。 |
| 提示 | 高亮 **一对** 当前可连的格子；若无则提示无解或抖动按钮。 |
